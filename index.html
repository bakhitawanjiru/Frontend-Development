From Concept to Code :

Join our Frontend Development Program!

From fundamentals to advanced techniques, learn web development with JavaScript and React.JS. Transform ideas into stunning websites and web apps, become a frontend engineer or web developer.

In today’s digital age, web development is a crucial skill that powers the internet and drives online experiences. With businesses and organisations increasingly relying on websites and web apps to reach/serve their audience, the demand for skilled web developers has never been higher. Learning web development equips you with the ability to create dynamic, responsive, and visually appealing web applications, enhancing user experiences and driving engagement. Whether you aim to launch a tech career, build your own projects, or freelance, mastering web development opens up a world of opportunities in the tech industry.

Zindua’s Frontend Development program is designed to take you from a beginner to a proficient frontend developer. The program covers essential technologies and tools, starting with HTML and CSS to build the structure and style of web pages. You will then dive into JavaScript, the backbone of interactive web development, learning how to create dynamic content and enhance user interactions. The curriculum culminates with React, a powerful JavaScript library for building user interfaces, enabling you to develop sophisticated, scalable, and high-performance web applications. Our hands-on approach ensures you gain practical experience through real-world projects and collaborative learning.

This program is perfect for anyone with a passion for technology and creativity, regardless of their previous experience. Aspiring web developers, designers looking to enhance their technical skills, and professionals seeking to transition into the tech industry will all benefit from this course. No prior coding experience is necessary; however, a basic understanding of computer operations will be helpful. Whether you’re a recent high school graduate, university graduate, career changer, or entrepreneur looking to create your own web presence, Zindua’s Frontend Development program provides the knowledge and skills to succeed in the dynamic field of frontend development.

Curriculum

What you’ll learn in our Frontend Development Program

Curated from our in-depth research and experience of the most in-demand skills for web development career pathways, our curriculum is broken down into 3 modules encompassing the following:

Learn the advanced JavaScript programming concepts to build more complex frontend apps. By the end of module, you’ll be able to build a web app with HTML, CSS, and vanilla JavaScript.
JavaScript Objects: Learn how to build and work with JavaScript objects with ES6 syntax
AJAX and Fetch API: Learn asynchronous JavaScript (AJAX) and Fetch API
Advanced JavaScript: Object-oriented programming and test-driven development
Deployment of Dashboards: Deploy dashboards with a coherent data story with Dash by Plotly

Learn the React library for frontend development. By the end of the module, you should be able to build and deploy a complete frontend application with React.JS and integrate with a NoSQL databases or Firebase.
React Foundations: Learn JSX and how to build React functional components Intermediate React: Event handling with React, props, and React state management Advanced React: Learn React hooks and custom hooks as well as testing Frontend Essentials: Learn how to work with Graph QL and how to deploy React Apps

All Zindua programs require students to be versed with computer programming fundamentals for building efficient code and scalable solutions. In this program, here are the essential data structures and algorithms you’ll learn: Basic Algorithms: Learn the Big-O Notation, Search, and Sorting Algorithms
Linear Data Structures: Learn Arrays, Hash maps, Stacks, Queues, Linked Lists, Trees, and Graphs
Non-Linear Data Structures: Trees, Graphs, and common algorithms involving trees/graphs such as pathfinding
Note that this is a Lite core program therefore, students will not get into advanced algorithms i.e. Dynamic Programming, and Greedy Algorithms which are covered in the longer/advanced Software Engineering Core.

To graduate the program, a student will be required to complete an end-to-end web development project involving building and deploying a web app with JavaScript and React.JS as well as connecting to a database or data source through GraphQL, REST API, or direct connections with a NoSQL database. Students have freedom to choose their final project based on their personal/career interests.
